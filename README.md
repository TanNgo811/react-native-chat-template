# react-native-chat-bar

React Native Chat Bar UI

## Installation

```sh
npm install react-native-chat-bar
```

## Usage

```js
import ChatBar from "react-native-chat-bar";

// ...

const result = await ChatBar.multiply(3, 7);
```
